Goal

Design and build front and back end for web portal

Build a RESTful API and database that allows for users with different roles (instructors/students) to login and access role specific pages. Design client side for ease of use and simplicity.

repo

Outcome

Functional web project with clean UI

Built SQL database and RESTful API with Express middleware for validation and authentication. API includes endpoints for instructors to login, create, and manage classes, and for students to login, browse, register/remove classes. I'm currently in the process of building the client-side of the portal.
